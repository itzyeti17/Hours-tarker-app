# Work Hours Tracker

A mobile app that allows users to track their work shifts and total hours worked.

## Tech Stack

* React Native (Expo)
* Node.js
* Express
* MongoDB Atlas

## Purpose

This project is being built as a portfolio project to practice full-stack mobile development. It is designed to help people easily track their work hours so they don’t lose track of the time they have worked.
